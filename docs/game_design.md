# 🎮 色块工厂 (Block Factory) - 游戏设计文档

## 📋 游戏概述

| 属性 | 值 |
|------|-----|
| **名称** | 色块工厂 (Block Factory) |
| **类型** | 2D 放置模拟经营 |
| **视觉风格** | 简单色块几何图形 |

## ⏰ 四、离线收益系统

### 4.1 离线收益公式

```
离线收益 = 每秒总产量 × 离线秒数 × 离线效率系数
```

| 参数 | 值 |
|------|-----|
| **最大离线时长** | 8 小时 (28,800 秒) |
| **离线效率系数** | 50% (0.5) |
| **最小离线时长** | 1 分钟才计算收益 |

### 4.2 离线收益示例

假设玩家每秒生产 1000 金币：

| 离线时长 | 计算 | 获得金币 |
|----------|------|----------|
| 30 分钟 | 1000 × 1800 × 0.5 | 900,000 |
| 1 小时 | 1000 × 3600 × 0.5 | 1,800,000 |
| 4 小时 | 1000 × 14400 × 0.5 | 7,200,000 |
| 8 小时 (上限) | 1000 × 28800 × 0.5 | 14,400,000 |

### 4.3 钻石加速选项

| 加速选项 | 消耗钻石 | 效果 |
|----------|----------|------|
| 立即领取 | 5 💎 | 立即获得全部离线收益 |
| 双倍离线 | 10 💎 | 下次离线效率 100% (持续 1 小时) |
| 延长上限 | 20 💎 | 离线上限 +2 小时 (本次有效) |

## 🔄 六、转生系统 (Prestige)

### 6.1 转生条件

| 条件 | 要求 |
|------|------|
| **最低金币** | 1,000,000 🪙 |
| **建筑要求** | 无 (可选) |
| **转生消耗** | 所有金币清零 |
| **保留内容** | 钻石、成就、总游戏时长 |

### 6.2 声望获取公式

```
声望获得 = floor(sqrt(金币 / 10,000))
```

**声望获取表示例：**

| 转生时金币 | 计算 | 获得声望 |
|------------|------|----------|
| 1,000,000 | sqrt(100) | 10 ⭐ |
| 5,000,000 | sqrt(500) | 22 ⭐ |
| 10,000,000 | sqrt(1000) | 31 ⭐ |
| 50,000,000 | sqrt(5000) | 70 ⭐ |
| 100,000,000 | sqrt(10000) | 100 ⭐ |
| 1,000,000,000 | sqrt(100000) | 316 ⭐ |

### 6.3 声望加成效果

| 声望效果 | 公式 | 说明 |
|----------|------|------|
| **生产加成** | 总产量 × (1 + 声望 × 1%) | 每点声望 +1% 产量 |
| **离线加成** | 离线效率 + (声望 × 0.1%) | 每点声望 +0.1% 离线效率 |
| **升级折扣** | 升级成本 × (1 - 声望 × 0.05%) | 每点声望 -0.05% 成本 (最高 50%) |

### 6.4 声望加成示例

| 总声望 | 生产加成 | 离线效率 | 升级折扣 |
|--------|----------|----------|----------|
| 0 ⭐ | +0% | 50% | 0% |
| 50 ⭐ | +50% | 55% | 2.5% |
| 100 ⭐ | +100% | 60% | 5% |
| 200 ⭐ | +200% | 70% | 10% |
| 500 ⭐ | +500% | 100% | 25% |
| 1000 ⭐ | +1000% | 150% | 50% (上限) |

## 🎯 八、核心公式汇总

### 8.1 生产相关

```
# 单个建筑实际产量
实际产量 = 基础产量 × (1 + 升级等级 × 0.1) × (1 + 声望 × 0.01)

# 总 DPS
总 DPS = Σ(每个建筑的实际产量)

# 考虑声望加成的总产量
总产量 = 基础总产量 × (1 + 声望 × 0.01)
```

### 8.2 成本相关

```
# 购买第 n 个建筑
购买成本 = base_cost × (1.5 ^ (n-1))

# 升级到下一级的成本
升级成本 = base_upgrade_cost × (1.5 ^ current_level)

# 带声望折扣的升级成本
实际升级成本 = 升级成本 × (1 - min(声望 × 0.0005, 0.5))
```

### 8.3 离线收益

```
# 基础离线收益
离线收益 = 总 DPS × min(离线秒数，28800) × 0.5

# 带声望加成的离线收益
实际离线收益 = 离线收益 × (1 + 声望 × 0.001)
```

### 8.4 转生相关

```
# 声望获得
声望 = floor(sqrt(金币 / 10000))

# 转生所需最小金币
最小转生金币 = 1,000,000
```
